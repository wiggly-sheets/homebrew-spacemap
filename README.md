# Homebrew tap for spacemap

Install spacemap via Homebrew:

```bash
brew tap wiggly-sheets/spacemap https://github.com/wiggly-sheets/homebrew-spacemap.git
brew install --cask spacemap
```

Or in one step:

```bash
brew install wiggly-sheets/spacemap/spacemap
```

## What is spacemap?

A macOS utility that visualizes yabai workspaces in a floating 2D grid overlay.

## Updating

When a new version is released, update the tap:

```bash
brew upgrade --cask spacemap
```

## Uninstalling

```bash
brew uninstall --cask spacemap
brew untap wiggly-sheets/spacemap
```
